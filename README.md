## Test

- Download the source code by `git clone `in your terminal

  ```shell
  git clone http://github.com/hannbusann/Polynomial_math
  ```

- Compile the testpolynomial.cpp using cmake

  ```
  mkdir build
  cd build
  cmake ..
  make
  ```

- execute the executable file, and view the outputs.

  ```
  ./testpolynomial
  ```
